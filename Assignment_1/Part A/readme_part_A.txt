This is a simple website for a coffee shop named Java Junction.

The website uses several different HTML controls to create a simple website for a coffee shop. 

The following tags are used in the website:

- <!DOCTYPE html>: This is the doctype declaration, which tells the browser that the document is an HTML5 document.
- <html>: This is the root element of the HTML document.
- <head>: This element contains meta information about the document, such as the title, links to CSS stylesheets, and a link to a favicon.
- <link rel="shortcut icon" href="coffee.ico" type="image/x-icon">: This link element is used to include a favicon for the website. The "href" attribute specifies the location of the favicon file, and the "rel" and "type" attributes specify that it is a shortcut icon.
- <link rel="stylesheet" type="text/css" href="style.css">: This link element is used to link to an external CSS stylesheet that can be used to style the website.
- <title>: This element specifies the title of the document, which is displayed in the browser's title bar or tab.
- <body>: This element contains the content of the HTML document.
- <header>: This element represents a container for introductory content or a set of navigational links.
- <h1>: This element represents a heading.
- <nav>: This element represents a section of a page that contains navigation links.
- <ul>: This element represents an unordered list.
- <li>: This element represents a list item.
- <a>: This element represents a hyperlink, which can be used to link to other web pages or specific locations within the same page.
- <h2>: This element represents a level 2 heading.
- <div>: This element is a container for flow content and can be used to group elements together for styling purposes.
- <img>: This element is used to embed images in the HTML document.
- <table>: This element represents a table.
- <tr>: This element represents a row in a table.
- <th>: This element represents a header cell in a table.
- <td>: This element represents a cell in a table.
- <button>: This element represents a clickable button.

The website includes a navigation menu to navigate to contact page, an introductory heading and an image of coffee cups.
It also includes a table that displays the menu items and their prices. The website also includes a subscribe button.
