This is a two pages website for a travel agency called Globe Trotters. 

The website uses several different HTML5 tags to create a presentable website for the agency.

The following tags have been used in the website:
- <!DOCTYPE html>: This is the doctype declaration, which tells the browser that the document is an HTML5 document.
- <html>: This is the root element of the HTML document.
- <head>: This element contains meta information about the document, such as links to CSS stylesheets and a link to a favicon.
- <link rel="stylesheet" type="text/css" href="style.css">: This link element is used to link to an external CSS stylesheet that can be used to style the website.
- <link rel="shortcut icon" href="favicon..ico" type="image/x-icon">: This link element is used to include a favicon for the website. The "href" attribute specifies the location of the favicon file, and the "rel" and "type" attributes specify that it is a shortcut icon.
- <title>: This element specifies the title of the document, which is displayed in the browser's title bar or tab.
- <body>: This element contains the content of the HTML document.
- <header>: This element represents a container for introductory content or a set of navigational links.
- <h1>: This element represents a heading.
- <nav>: This element represents a section of a page that contains navigation links.
- <ul>: This element represents an unordered list.
- <li>: This element represents a list item.
- <a>: This element represents a hyperlink, which can be used to link to other web pages or specific locations within the same page.
- <section>: This element represents a thematic grouping of content.
- <h2>: This element represents a level 2 heading.
- <article>: This element represents a self-contained composition in a document.
- <h3>: This element represents a level 3 heading.
- <img>: This element is used to embed images in the HTML document.
- <p>: This element represents a paragraph of text.
- <footer>: This element represents a container for the footer of a document or section.

The first page serves as the homepage and includes a header with a navigation menu, a section displaying popular destinations, and a footer with links to various pages of the website. The second page is the about us page which describes the company and its services.
The website also includes a favicon.
